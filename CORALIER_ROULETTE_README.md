# 🎰 코랄리에 룰렛 게임

**미니언즈런 행사용 코랄리에 브랜드 홍보 룰렛 게임**

React 기반 커스터마이징 가능한 룰렛 게임 애플리케이션입니다.

---

## ✨ 주요 기능

### 🎯 코랄리에 맞춤 디자인
- **브랜드 색상**: 연핑크, 민트 그라디언트
- **상단 배너**: 코랄리에 알파벳 로고 (헤더)
- **룰렛 중앙**: 코랄리에 심볼 마크
- **9:16 세로 비율**: 운동장 텐트에서 모니터로 최적화

### 🎁 기본 경품 설정
- **1등 (10%)**: 치약,칫솔,구강스프레이(2+1)세트
- **2등 (40%)**: 구강스프레이 단품
- **3등 (50%)**: 마우스워시 단품

### ⚙️ 설정 메뉴 (햄버거 버튼)
- **경품 이름 변경**: 텍스트 직접 수정 가능
- **비율 조정**: 각 항목의 당첨 확률 변경
- **색상 변경**: 각 섹션 색상 커스터마이징
- **항목 추가/삭제**: 경품 개수 자유롭게 조절
- **자동 조정**: 비율 합계를 100%로 자동 보정
- **초기화**: 기본 설정으로 복원

### 🎵 음악 시스템
5가지 음악이 상황에 맞게 자동 재생:

1. **waiting.mp3** - 대기 음악 (룰렛 돌리기 전, 루프)
2. **spinning.mp3** - 룰렛 회전 음악 (5초)
3. **prize1.mp3** - 1등 당첨 효과음
4. **prize2.mp3** - 2등 당첨 효과음
5. **prize3.mp3** - 3등 당첨 효과음

> 📁 음악 파일은 `public/audio/` 폴더에 넣으세요
> 자세한 내용: `public/audio/AUDIO_GUIDE.md`

### 📱 사용 환경
- ✅ 운동장 몽골텐트 앞
- ✅ 노트북/태블릿 → 대형 모니터 연결
- ✅ 9:16 세로 비율 (모바일 화면 비율)
- ✅ 터치 지원

## 🖼️ 로고 이미지 준비 (필수!)

### 필요한 파일

1. **logo-banner.png** (상단 헤더용)
   - 위치: `public/images/logo-banner.png`
   - 권장 크기: 가로 400px 이상
   - 형태: 가로로 긴 알파벳 로고
   - 배경: 투명 PNG 권장

2. **logo-symbol.png** (룰렛 중앙용)
   - 위치: `public/images/logo-symbol.png`
   - 권장 크기: 200x200px (정사각형)
   - 형태: 코랄리에 심볼 마크
   - 배경: 투명 PNG 권장

### 다운로드 방법

쿠팡 이미지: https://image11.coupangcdn.com/image/cmg/oms/banner/1433d540-8aaf-4ce8-9d6a-4c9b14acb5cd_270x270.png

1. 위 링크에서 이미지 다운로드
2. 포토샵/Canva로 배너용(가로)과 심볼용(정사각형) 2개 제작
3. `public/images/` 폴더에 저장

> 자세한 가이드: `public/images/LOGO_GUIDE.md`

## 🚀 시작하기

### 1. 패키지 설치

```bash
npm install
```

### 2. 로고 이미지 추가

`public/images/` 폴더에 다음 2개 파일 넣기:
- logo-banner.png
- logo-symbol.png

### 3. 음악 파일 추가 (선택)

`public/audio/` 폴더에 다음 5개 파일 넣기:
- waiting.mp3
- spinning.mp3
- prize1.mp3
- prize2.mp3
- prize3.mp3

### 4. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 표시되는 주소로 접속 (보통 `http://localhost:5173`)

### 5. 프로덕션 빌드

```bash
npm run build
```

빌드된 파일은 `dist/` 폴더에 생성됩니다.

## 🎮 사용 방법

### 행사 당일 (미니언즈런)

1. **노트북/태블릿 준비**
   - `npm run dev` 또는 빌드된 파일 실행
   - 대형 모니터에 연결
   - 세로 모드로 설정 (9:16)

2. **룰렛 돌리기**
   - 참가자가 화면 터치 또는 클릭
   - "룰렛 돌리기" 버튼 누르기
   - 5-8바퀴 회전 후 자동 정지
   - 당첨 결과 팝업 표시

3. **경품 지급**
   - 1등: 치약,칫솔,구강스프레이(2+1)세트
   - 2등: 구강스프레이 단품
   - 3등: 마우스워시 단품

4. **비율 조정 (필요시)**
   - 우측 상단 햄버거 버튼 클릭
   - 비율 조정 (예: 1등 5%, 2등 30%, 3등 65%)
   - "저장" 버튼 클릭

## 🎨 커스터마이징

### 경품명 변경

1. 햄버거 메뉴 열기 (우측 상단)
2. 각 항목 이름 직접 수정
3. "저장" 클릭

또는 코드로 변경:

`src/App.jsx` 파일의 `DEFAULT_PRIZES` 배열 수정:

```javascript
const DEFAULT_PRIZES = [
  { id: 1, name: '특별상', percentage: 5, color: '#FF69B4' },
  { id: 2, name: '우수상', percentage: 20, color: '#7FFFD4' },
  { id: 3, name: '참가상', percentage: 75, color: '#FFB6C1' }
]
```

### 색상 테마 변경

`src/App.css` → 배경 그라디언트 수정
`src/components/Roulette.css` → 버튼 색상 수정

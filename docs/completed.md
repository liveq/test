## 기술 스택 분석

### 가장 많이 사용한 라이브러리
1. **Canvas API** (7개) - 이미지 처리 필수
2. **JSZip** (5개) - 일괄 다운로드용
3. **순수 JS** (5개) - 간단한 도구는 라이브러리 불필요
4. **pica** (3개) - 고품질 리사이즈

### CDN 활용
- **jsdelivr:** 모든 라이브러리 (빠르고 안정적)
- **unpkg:** 대체 CDN (백업용)

# 음악 파일 가이드

이 폴더에 다음 5개의 음악 파일을 넣어주세요:

## 필요한 파일

1. **waiting.mp3** - 대기 음악 (룰렛을 돌리기 전 배경음악, 자동 루프)
2. **spinning.mp3** - 룰렛 회전 음악 (룰렛이 돌아갈 때 재생)
3. **prize1.mp3** - 1등 당첨 효과음
4. **prize2.mp3** - 2등 당첨 효과음
5. **prize3.mp3** - 3등 당첨 효과음

## 파일 형식

- MP3 형식 권장
- 파일명은 정확히 위와 동일해야 합니다
- 파일이 없어도 에러는 발생하지 않으며, 단순히 재생되지 않습니다

## 음악 재생 흐름

1. 앱 시작 → `waiting.mp3` 자동 재생 (루프)
2. 룰렛 클릭 → `waiting.mp3` 정지, `spinning.mp3` 재생
3. 룰렛 정지 → `spinning.mp3` 정지
4. 결과 표시 → 등수에 맞는 효과음 재생 (`prize1.mp3`, `prize2.mp3`, `prize3.mp3`)
5. 3초 후 → `waiting.mp3` 재개

## 추천 음악 길이

- waiting.mp3: 30초~1분 (루프되므로 짧아도 OK)
- spinning.mp3: 5~7초 (룰렛 회전 시간과 맞춤)
- prize1/2/3.mp3: 2~5초 (짧은 효과음)

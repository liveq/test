## 프로젝트 개요

BAAL은 사용자들이 브라우저에서 바로 사용할 수 있는 무료 온라인 도구를 제공하는 프로젝트입니다.
회원가입 불필요, 100% 클라이언트 사이드 처리로 개인정보 보호가 보장됩니다.

**핵심 가치:**
- 무료 & 광고 없음 (초기)
- 빠른 속도 (서버 없음)
- 개인정보 보호 (로컬 처리)
- 모바일 최적화
- 다크모드 지원
- 한글/영어 지원

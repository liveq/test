# 🎰 룰렛 게임

React 기반 커스터마이징 가능한 룰렛 게임 애플리케이션입니다.

## ✨ 주요 기능

### 🎯 룰렛 게임
- **비율 설정**: 기본 10% (1등), 40% (2등), 50% (3등)
- **당첨 확률**: 설정한 비율에 따라 자동 계산
- **부드러운 애니메이션**: 5-8바퀴 회전 후 정확한 위치에 정지
- **시각적 피드백**: 당첨 결과를 애니메이션과 함께 표시

### ⚙️ 설정 메뉴
- **햄버거 메뉴**: 우측 상단 버튼으로 설정 패널 열기
- **비율 조정**: 각 항목의 당첨 비율을 자유롭게 변경
- **항목 편집**: 이름, 색상 변경 가능
- **항목 추가/삭제**: 원하는 만큼 항목 추가 및 제거
- **자동 조정**: 비율 합계를 100%로 자동 조정 기능
- **초기화**: 기본 설정(1등 10%, 2등 40%, 3등 50%)으로 복원

### 🎵 음악 시스템
5가지 음악이 상황에 맞게 자동 재생됩니다:

1. **대기 음악** (`waiting.mp3`): 앱 시작 시 자동 재생 (루프)
2. **회전 음악** (`spinning.mp3`): 룰렛 회전 시 재생
3. **1등 효과음** (`prize1.mp3`): 1등 당첨 시 재생
4. **2등 효과음** (`prize2.mp3`): 2등 당첨 시 재생
5. **3등 효과음** (`prize3.mp3`): 3등 당첨 시 재생

> 📁 음악 파일은 `public/audio/` 폴더에 위 파일명으로 저장하세요.
> 자세한 내용은 `public/audio/AUDIO_GUIDE.md` 참조

### 📱 반응형 디자인
- **9:16 비율**: 모바일 세로 화면에 최적화
- **데스크탑 지원**: 노트북, 데스크탑에서도 정상 작동
- **모니터 연결**: 핸드폰/노트북을 외부 모니터에 연결해 사용 가능

## 🚀 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 표시되는 로컬 주소로 접속하세요 (보통 `http://localhost:5173`)

### 프로덕션 빌드

```bash
npm run build
```

빌드된 파일은 `dist/` 폴더에 생성됩니다.

### 프리뷰

```bash
npm run preview
```

## 🎨 커스터마이징

### 기본 비율 변경

`src/App.jsx` 파일의 `DEFAULT_PRIZES` 배열을 수정하세요:

```javascript
const DEFAULT_PRIZES = [
  { id: 1, name: '1등', percentage: 10, color: '#FFD700' },
  { id: 2, name: '2등', percentage: 40, color: '#C0C0C0' },
  { id: 3, name: '3등', percentage: 50, color: '#CD7F32' }
]
```

### 색상 테마 변경

- **배경**: `src/index.css`의 `#root` 배경 gradient 수정
- **앱 배경**: `src/App.css`의 `.app` 배경색 수정
- **버튼 색상**: `src/components/Roulette.css`의 `.spin-button` 수정

## 🛠️ 기술 스택

- **React 18** - UI 라이브러리
- **Vite** - 빌드 도구
- **CSS3** - 스타일링 (애니메이션 포함)
- **SVG** - 룰렛 휠 렌더링
- **Web Audio API** - 음악 재생

## 💡 사용 팁

1. **음악 파일 준비**: 게임 시작 전 `public/audio/` 폴더에 5개의 음악 파일을 넣으세요
2. **비율 조정**: 햄버거 메뉴를 열어 원하는 비율로 조정 가능
3. **항목 추가**: "항목 추가" 버튼으로 4등, 5등 등 원하는 만큼 추가 가능
4. **자동 조정**: 비율 합계가 100%가 아니면 "100%로 조정" 버튼 클릭
5. **모바일 사용**: 9:16 비율로 최적화되어 있어 세로 모드가 가장 보기 좋습니다
